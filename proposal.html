Sneaker Sales Manager

Problem Description

The topic of interest is sneaker reselling. Sneaker reselling is the idea of purchasing limited shoes such as newly released Jordan’s and selling them at a higher price which has been coined the term ‘flipping’. Sneaker resellers make many transactions which causes the problem of losing track of what sneakers they’ve purchased and sold if they aren’t using something to manage their sales. Not managing sneaker transactions can negatively impact resellers because they will lose valuable information such as details about sneakers they’ve bought and sold and how much profit/losses they’ve made.

Sneaker sales management is a problem that needs to be addressed because current and previous attempts at solving this problem are insufficient in tracking and sorting one’s sneaker sales. For example, popular sites like StockX's portfolio feature allows the user to manage sneakers but only shows how much the shoes currently sell for rather than how much users sold them for. Another web app that is aimed at managing sneaker sales, SneakerBills , includes information on sale history. However, this application lacks the ability of sorting sneakers by size, color, and even product name.

App Description

The users of the sneaker sales management application are sneaker resellers. The app is intended to help manage sneaker sales for both novice resellers with few amounts of sales and experienced resellers with a lot of sales. The application is intended for data collection so users will be viewing their own information.

Functionality

Users will be able to add and sort through their sneaker sales. Interactions that will be provided include:

  • Inputting sneaker sale information
  • Searching for a specific sneaker
  • Editing details about sneakers
  • Sorting through sneaker sales ex) size, color

This application is a solution to managing sneaker sales because it allows users to input information about sales history. Unlike other existing sneaker sales management applications, users will now be able to sort through sales in more detail. Overall, the application will make managing and sorting sneaker sales easier for resellers.
